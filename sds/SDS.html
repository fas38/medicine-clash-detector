1. Architectural Pattern

For this project, we are using Observer pattern within Model View Controller (MVC) pattern. It has four parts:

  • Model
  • View
  • Controller
  • Observer





  • The Model helps to handle database. It is a data access layer that handles the data.
  • View represents the visualization of the data that model contains.
  • Controller acts on both model and view. It controls the data flow into model object and updates the view whenever data changes. It keeps view and model separate.
  • By implementing Observer interfaces we can eliminate coupling between the Model and View.

2. Component Interactions

Our selected platforms are IOS and Android. For android app,we are going to use kotlin and for ios app, we are going to use swift. We are also using Google's "Mobile Vision API" to read text from images.

In database,we are using only local database. So online connection is not required to run both the IOS and Android version of the app.

3.2 Details of Class

  • UserInfo

    UserInfo class has name, username, password, email, image,user_id attributes.

  • ClashReport

    ClashReport class has medicine_1, medicine_2, clash_reason, user_id attributes.

  • DrugList

    DrugList class has drug_name,user_id attributes.

  • MedicalCondition

    MedicalCondition class has type, prohibited_drugs, user_id attributes.
    It has also four sub-class:

    • Heart Disease: This sub-class is for heart disease condition.
    • Kidney Disease: This sub-class is for kidney disease condition.
    • High Blood Pressure: This sub-class is for high blood pressure condition.
    • Diabetes: This sub-class is for diabetes related condition.

  • Medicine

    Medicine class has name, generic_name, brand_name, precaution, incompitable_drugs attributes.
    It has also four sub-class:

    • Antipyretics: This sub-class is for drugs related to reducing fever.
    • Analgesics: This sub-class is for drugs related to reducing pain.
    • Antibiotics: This sub-class is for drugs related to inhibiting germ growth.
    • Stimulants: This sub-class is for drugs related to stimulants.
